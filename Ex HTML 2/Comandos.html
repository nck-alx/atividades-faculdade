    • #1 - Como fazer citações

    Para destacar uma frase, parágrafo ou versículo como citação, utilizamos a tag "q". Exemplo:

    O que não provoca minha morte faz com que eu fique mais forte.
    - Friedrich Nietzsche






    #2 - Como abreviar textos

    Para criar a explicação de uma abreviação de um termo, utilizamos a tag "abb title=''", sendo o parênteses usado para o que aparecerá quando o usuário posicionar o cursor sobre o texto. Exemplo:

    "O HTML é utilizado por vários programadores..."






    #3 - Como inverter um texto

    Para criar um Texto Invertido devemos usar a tag "bdo dir="rtl"". Exemplo:

    Socorram-me, subi no ônibus em Marrocos.




    #4 - Como criar listas ordenadas

    Utilizamos a tag "ol" para definir como deverá ser feita a enumeração dos itens através do comando type="", e a tag "li" para definir o começo e o término dos itens da lista. Exemplo:

      Top 5 vilões da DC
    1. Coringa
    2. Brainiac
    3. Darkseid
    4. Flash Reverso
    5. Exterminador




    #5 - Como criar listas não ordenadas

    Utilizamos as tags "ul" para definir o símbolo que enumera os itens da lista (Disco, quadrado, ou círculo), e a tag "li" para definir o começo e o término dos itens da lista. Exemplo:

      Top Vilões da Marvel
    • Magneto
    • Thanos
    • Doutor Destino
    • Galactus
    • Rei do Crime




    #6 - Como deixar um texto em negrito

    Utilizamos a tag "n" ou "strong" para destacá-lo em negrito. Exemplo:



    "Este texto em negrito deixará de ficar destacado a partir do momento em que a frase possuir um símbolo especial, como por exemplo um arroba, uma vírgula ou underline"




    #7 - Como deixar um texto em itálico

    Utilizamos a tag "i" para destacá-lo em itálico. Exemplo:



    "Este texto deixará de ficar destacado a partir do momento em que a frase possuir um símbolo especial, como por exemplo um arroba, uma vírgula ou underline"




    #8 - Como deixar um texto marcado

    Utilizamos a tag "mark" para destacá-lo, utilizando o comando style="background-color: ;" para alterar a cor do marcador. Exemplo:



    "Este texto deixará de ficar destacado a partir do momento em que a frase possuir um símbolo especial, como por exemplo um arroba, uma vírgula ou underline"






    #9 - Como formatar um texto (títulos)

    Utilizamos as tags "h1, h2, h3, h4, h5, h6" para formatar um texto em títulos e sub-títulos. Exemplos:


    título 1

    título 2

    título 3

    título 4

    título 5
    título 6





    #10 - Como criar quebras de linha e linhas horizontais

    Utilizamos a tag "br" para criar uma quebra de linha, e a tag "hr" para criar uma linha horizontal (quebra de título). Exemplos:


    Título

    A quebra de linha será feita quando uma frase se encerrar.
    Entretanto, também poderá ser feita quando existir uma palavra com hífen, como por exemplo "guarda-chuva"
    , "guarda-roupas", dentre outras.
